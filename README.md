# Plattform Industrie 4.0

Dieses Repository ist **kein einzelnes Projekt**, sondern ein **Sammel- und Forschungs-Hub** zu Themen rund um Industrie 4.0, OT/IT-Sicherheit, Künstliche Intelligenz und regulatorische Compliance.

Aktuell gibt es **ein aktives Hauptprojekt**:

**ZTA+KI 2026**  
Zero Trust Architektur mit KI-Assistenz für sichere, konforme und resiliente Industrie-4.0-Systeme  
→ [**Zum Projekt-Zweig**](/tree/main/ZTA%2BKI_2026)

## Was steht wo?

- **Word-Dokument / Publikation**  
  → Motivation, strategischer Überblick, Kernprozess, Ergebniszusammenfassung, Empfehlungen für KMU  
  → bewusst schlank und lesbar gehalten

- **GitHub (dieses Repo)**  
  → Detaillierte Umsetzung, Rohdaten, 30+ Seiten Prompt-Analysen, JSON-Beispiele, Abweichungsdetektion, SoA-Mappings, interaktive Diagramme  
  → alles, was zu umfangreich, technisch oder versionsabhängig ist, um ins gedruckte Dokument zu passen

- **Zukünftige Projekte**  
  Derzeit gibt es nur **ein** ausgearbeitetes Projekt (ZTA+KI 2026).  
  Weitere Themen (z. B. Agentic AI in OT, Reasoning für Security-Analysen, RAG für KMU-Compliance) sind geplant und werden als eigene Zweige oder Ordner hinzukommen.
